# Reference Selection

Use this reference when selecting bibliography entries for a Chinese undergraduate thesis.

## Priority

School template, task book, advisor requirements, and department notices override all defaults here.

## Default Heuristics

When no school-specific requirement exists:

- Prefer recent, verifiable sources, but do not discard authoritative older sources solely by year.
- Keep the Chinese/English ratio aligned with the thesis topic and available verified literature.
- Treat total count as a target from the school or advisor, not a fixed number.
- Prefer sources with DOI, publisher pages, journal pages, university database records, or other stable verification paths.

## Hard Rules

- Do not fabricate references.
- Do not use a source when title, authors, year, venue, or URL/DOI cannot be verified.
- Mark uncertain candidates as `needs_check` instead of citing them as confirmed.
- Generate or update the reference verification checklist before final delivery.
