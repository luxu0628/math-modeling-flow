# 默认论文格式

在不套用学校模板时，DOCX 生成器使用以下内置格式。该格式用于生成可阅读、可复制到学校模板中的正文初稿，不声明已经符合某一学校的最终版式。

- 论文题目：黑体 18pt，加粗，居中
- 一级标题：黑体 18pt，加粗
- 二级标题：黑体 15pt，加粗
- 三级标题：黑体 12pt，加粗
- 中文正文：宋体 12pt
- 英文正文：Times New Roman 12pt
- 正文段落：1.25 倍行距，首行缩进约 2 字符
- 摘要、Abstract、目录、参考文献、致谢、附录标题居中
- 中文关键词：标签黑体 12pt 加粗，内容宋体 12pt
- 英文关键词：标签 Times New Roman 12pt 加粗，内容 Times New Roman 12pt
- 图题：宋体 10.5pt，图下方居中，单倍行距
- 表题：宋体 10.5pt，表上方居中，单倍行距
- 表格文字：宋体 10.5pt，居中
- 公式：默认保留 LaTeX/source 文本；用户选择 `formula_image` 时，可按 image map 插入公式图片
- 参考文献正文：宋体 10.5pt，悬挂缩进约 2 字符
