# 高分论文范式（从真实国赛优秀论文提炼）

> 来源：研读 13 篇公开渠道的国赛获奖论文——2021 年官方展示优秀论文（论文编号 A028/A115/A217/B026 摘要精读，
> C085/C169/D017/D026/D034/E025/E037 正文研读，见教育部"中国大学生在线"论文展示专题）；
> 以及《工程数学学报》刊载的特等奖论文（2003 SARS 传播·厦门大学、2008 数码相机定位·北京航空航天大学，
> 刊载版含"编者按"=专家点评视角）。
> 本文件是 `math-modeling-flow` 的 G2/G3/G4/G5 叠加规则：**学形不抄实**。

## 一、摘要硬清单（G5 写摘要逐条对照）

1. **标题 = 方法 + 对象**：如《基于双因素方差分析的乙醇制备C4烯烃的研究》。特等奖标题更学术化
   （《透视投影中圆心投影定位和双目立体系统标定》——完全不提赛题口语名）。
2. **总起一段**：背景 + 全文总方法路线，3–5 行。
3. **"针对问题N"逐段展开**，每问三件套：**模型名（显式命名）→ 怎么做（关键步骤/算法点名）→ 量化结果**。
4. **量化结果直接进摘要**：方程、坐标、百分比、最优参数（350℃、RMS 3.0mm、接收比 1.103%…），
   禁止"得到了较好的结果"这类空话。
5. **题目要求的交付物在摘要点名**：如"结果保存于 result1.xlsx"。
6. **社会/应用类题**：摘要末尾给"对实际的建议/应用价值"一句（SARS 篇给防控建议与经济影响）。
7. **关键词 5 个左右**：对象 + 核心方法（坐标变换、蒙特卡洛、BFGS…）。

## 二、正文结构范式

- 编号章节骨架：**问题重述 → 问题分析（逐问 2.1/2.2…）→ 模型假设与符号说明 → 逐问建模与求解
  （每问内：建模→求解→结果→结果分析）→ 检验 → 模型评价**。
- **问题分析必须逐问写"为什么这么建模"**，不是复述题目。
- **假设 4–6 条**短句，每条服务于后续建模；**符号表用三线表**（符号｜含义）。
- 优化/方案类：**最优方案用表格完整呈现**；分类讨论要穷举并编号（类型1/类型2…）。
- 数据类：方法 → 指标定义（公式）→ 调参图（如轮廓系数-K 肘部图）→ 工具点名（"使用 python 编程计算"）→
  结果三线表 → **紧跟"结果分析"小节**（排名大表 + 占比饼图 + 分析）。

## 三、图表与证据范式

1. 图表全部**编号 + 中文标题**（图4 K-means 轮廓系数图；表2 最优切割方案）。
2. 结果表用**三线表**；占比用饼图；调参/收敛过程必须配曲线图。
3. **每个结果图表后面必须有分析文字**，不许"贴图不说话"。
4. 程序可运行的证据：附录可放**运行输出截图**——与 G3"结果落盘冻结"同源：图与数必须来自真实运行。

## 四、结果与检验范式（评委实际在看什么）

1. **自定义指标必须先定义后使用**：2021A 三篇获奖论文的"接收比"为 67.46% / 1.53% / ≈1.1%，
   差几个数量级却都获奖——绝对值取决于口径，评委看**定义清晰 + 计算自洽 + 对照结论方向一致**。
2. **永远有对照组**：调节前 vs 调节后、基准 vs 优化后；结论落在"提升了多少"。
3. **第二方法互验是最高级检验**：2021A 三篇用不同方法（几何构造/纯优化/降维+BFGS）收敛到同一张抛物面
   （焦准参数 561.7–562.4、顶点下移≈0.3m）——有余力就用第二种方法粗算验证主结果（G3 规则）。
4. 误差/灵敏度分析嵌在每问求解末尾（关键词里点名"敏感性分析"）。
5. 迭代类算法要给**收敛证据**（如 RMS 4.2→3.0mm 且迭代中保持稳定）。

## 五、同题路线自由（G2 规则）

同一道 FAST 题（2021A），几何构造、纯优化叙事、降维+经典数值算法三条路线全部获奖。
**G2 不要猜"出题人想要什么标准答案"，选团队最能做扎实、最能自圆其说的路线**；
路线选定后全文叙事要统一（如全程"优化"语言、模型 I/II 编号清晰）。

## 六、特等奖（期刊版）额外特征

- 篇幅精炼（6–11 页期刊版），**原理性贡献突出**（数码相机定位篇用 Cauchy 不等式做严格证明）。
- "编者按"=专家点评，点出获奖理由——写作时可自问：**如果给本文写编者按，亮点一句话是什么？**
  答不出来 = 论文没有记忆点。

## 样本局限

- 样本 13 篇，规律以"多篇同构"为准入标准，孤例不入；
- 流传的扫描版论文集普遍存在页序错乱、无文字层等问题，研读时注意核对页码；个别数字以原文为准。
